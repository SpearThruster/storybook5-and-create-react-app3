# Storybook 4 demo using create-react-app 3.0 + TypeScript (using Babel)

A complete working setup using [cra 3](https://github.com/facebook/create-react-app) with `--typescript`, storybook 5.
# Launching Storybook

- `npm i` (first time)
- `npm run storybook`
